# Vunnam Akash Reddy - Personal Portfolio

A modern, responsive, and clean developer portfolio built with React, Tailwind CSS, and Vite. Designed to showcase projects, skills, and experience with a minimalist and professional approach.

## 🚀 Features
- **Responsive Design**: Flawlessly optimized for Mobile, Tablet, Laptop, and Desktop screens.
- **Dark Mode**: Fully supported light/dark mode with system preference detection.
- **Smooth Animations**: Integrated with Framer Motion for clean hover states, scrolling, and a custom loading screen.
- **SEO Optimized**: Pre-configured metadata and descriptions to achieve a high Lighthouse score.


## 🛠 Tech Stack
- **Framework**: React.js (Vite)
- **Styling**: Tailwind CSS (v4)
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Scrolling**: React Scroll

## 📁 Folder Structure
```text
├── public/                 # Static assets (images, logos, CV)
│   ├── logo.png            # Main brand logo
│   └── assets/             # Project screenshots
├── src/                    
│   ├── components/         # Reusable UI components (Hero, Navbar, Projects, etc.)
│   ├── App.jsx             # Main layout and routing
│   ├── index.css           # Tailwind configurations and global styles
│   └── main.jsx            # Application entry point
└── package.json            # Dependencies and scripts
```

## 💻 Local Development

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```


## 📊 Performance & SEO
- Lighthouse audited to ensure accessibility, best practices, and SEO optimization.
- Lazy loaded components and compressed static assets for blazing-fast performance.
